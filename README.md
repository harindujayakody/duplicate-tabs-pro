## ⚡ Key Features

*   **Zero-Latency Duplication:** Duplicate active webpages instantly without slowing down your browser.
*   **Custom Keyboard Shortcuts:** Bind your own custom hotkeys (default: `Ctrl+Shift+K` / `MacCtrl+Shift+K`) to duplicate tabs instantly from anywhere.
*   **Premium Pure Black Theme:** Enjoy a modern light mode or an ultra-sleek, battery-saving Pure Black (`#000000`) dark mode.
*   **Active Status dot:** Visual indicator confirming that the background worker is running and ready.
*   **Built-in Client Docs:** In-depth keyboard configuration tools and client documentation hosted directly inside the extension's settings page.
*   **Privacy First:** Collects zero user data and requires only the minimal `"activeTab"` security permission.

## 🛠️ Local Installation (Developer Mode)

To run the extension locally on your computer:

1.  **Clone the Repository:**
    ```bash
    git clone https://github.com/harindujayakody/duplicate-tabs-pro.git
    ```
2.  **Open Chrome Extensions Page:**
    Open Google Chrome and navigate to `chrome://extensions/` in your address bar.
3.  **Enable Developer Mode:**
    Toggle the **Developer mode** switch in the top-right corner to **ON**.
4.  **Load the Extension:**
    Click **Load unpacked** in the top-left corner and select the repository directory on your computer.
5.  **Pin the Extension:**
    Click the puzzle piece icon in the Chrome toolbar and pin **Duplicate Tabs Pro** for quick access.

---

## ⌨️ Customizing the Shortcut

1.  Open the extension popup and click **Settings & Help Docs** (or open the extensions page settings).
2.  Click the **Customize Shortcuts Page** button.
3.  Scroll down to **Duplicate Tabs Pro** and enter your preferred key combination.
4.  *Note: After changing shortcuts, refresh any open tabs or restart Chrome to apply the new hotkey listeners.*
