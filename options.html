Client Documentation & Help

How to Duplicate Your Tab

  1. Method 1: Click the Action Icon

    Click the Duplicate Tabs Pro icon in your browser extension bar. Inside the popup, click the large blue "Duplicate Current Tab" button.

  2. Method 2: Use the Global Keyboard Command

    Press the assigned keyboard shortcut (default is Ctrl+Shift+K) on any active tab. The page will immediately clone in a new tab adjacent to the current one.

Troubleshooting Guide

  • Why does the shortcut not work on some pages?

    For security, Google Chrome disables all extensions on protected system pages (like chrome://settings, chrome://extensions, the New Tab Page, or the official Chrome Web Store). On these pages, duplication via shortcut will not trigger.

  • The shortcut isn't working on normal websites?

    Verify that another extension isn't conflicting with your shortcut. Click the Customize Shortcuts Page button in the left panel to check if the shortcut is mapped correctly or if Chrome reports a conflict.

  • How do I change the Dark/Light theme?

    Use the toggle in the top-right of this page, or click the theme selector inside the browser toolbar popup. The setting will sync instantly across both pages.
